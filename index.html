  • git config --global user.name blabla - lets you to set your username
  • git config --global user.email blabal@bla - lets you to set your email
  • git init - initializes a new (yet local) repository on your pc
  • git status - lets you see if any changes were made in your rep
  • git add - adds new or changed files to Git staging area from your local working directory
  • git commit - creates a snapshot of your rep at specific time
  • git log - lets you see the commit history
  • git checkout - lets you navigate between the branches
  • git branch - lets you create, list, rename, delete branches
  • git push - uploads local repository content to a remote repository
  • git pull - fetches and downloads content from a remote repository and updates your local to match that content
  • git merge - basically merges new independent branches with main one
  • git clone - copies an existing repository or branch within it
  • gitignore - specifies intentionally untracked files that Git should ignore
  • git revert - creates new commit which undoes previous committed changes
  • git reset - undoes last commit and uncomitted changes
  • git rebase - allows ypu to integrate changesfrom one branch onto another
  • git cherry-pick - choosing commit from one branch and applying it to another one
  • git stash - takes uncomitted changes and saves them for later use
